# 实现数据互通

本章目标：让 Rust、C++/Python 节点围绕同一套消息模型协作。

## 1. 透明消息用于互通

保持 `SensorData` / `MotorCommand` 为透明类型：

```rust
#[roplat::roplat_msg]
#[repr(C)]
pub struct SensorData { /* ... */ }
```

这是跨语言互通的基础。

## 2. 引入 opaque（可选）

当你确实需要语言私有数据时，再引入：

```rust
#[roplat::roplat_msg(lang = "cpp")]
pub struct CppData;
```

注意：

1. Rust 侧不填字段。
2. 字段由 `cpp/src/cpp_data.h` 或目标语言用户文件定义。

## 3. 数据流建议

1. 主链路使用透明消息。
2. 私有优化状态使用 opaque。
3. 主链路先稳定，再追加 opaque。
