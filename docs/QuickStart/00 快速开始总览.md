# 快速开始总览

本部分是完整施工手册：从空目录开始，到可运行的 Rust 项目，再扩展到 C++/Python 节点和数据互通。

## 前提假设

1. 你不需要先获取 roplat 源码。
2. roplat 与 roplat_build 已作为库发布。
3. 你可以通过 `cargo add` 安装依赖。

## 章节路线

1. [从零创建 Rust 项目](01%20从零创建%20Rust%20项目.md)

4. [实现数据互通](04%20实现数据互通.md)
5. [编译与发布使用](05%20编译与发布使用.md)
