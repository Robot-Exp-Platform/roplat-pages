# RoPlat

世人苦 `ROS` 已久，于是 `RoPlat` 诞生了。

Roplat 是一个面向运行期安全的机器人操作系统。它希望把常见运行期问题尽可能前移到编译期，帮助你在机器人实验和工程中获得更稳定、可预测的行为。
