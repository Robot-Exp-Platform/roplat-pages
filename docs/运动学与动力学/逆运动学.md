# 不同自由度下的逆运动学

## 1. **2自由度平面机械臂**

### 1.1 几何法公式

**构型**：两旋转关节，平面运动（XY平面）  
**末端坐标**：\( (x, y) = (l_1\cos\theta_1 + l_2\cos(\theta_1+\theta_2), \ l_1\sin\theta_1 + l_2\sin(\theta_1+\theta_2)) \)  

**逆解公式**：  

1. 计算目标点到原点的距离：  
   \[
   r = \sqrt{x^2 + y^2}
   \]  
   **存在解条件**：\( r \leq l_1 + l_2 \)  

2. 余弦定理求解 \(\theta_2\)：  
   \[
   \theta_2 = \pm \arccos\left( \frac{x^2 + y^2 - l_1^2 - l_2^2}{2l_1l_2} \right) \quad (\text{双解：肘部上/下})
   \]  

3. 求解 \(\theta_1\)：  
   \[
   \theta_1 = \arctan2(y, x) - \arctan2\left( l_2\sin\theta_2, l_1 + l_2\cos\theta_2 \right)
   \]

## 2. **3自由度球坐标机械臂**

### 2.1 Paul代数法（RRR构型）

**末端位置**：\( o_3 = [x, y, z]^T \)  
**关节变量**：\( \theta_1, \theta_2, \theta_3 \)  

**逆解步骤**：  

1. **求解 \(\theta_1\)**（绕Z轴旋转）：  
   \[
   \theta_1 = \arctan2(y, x)
   \]  

2. **求解 \(\theta_2\)**（绕Y轴俯仰）：  
   \[
   r = \sqrt{x^2 + y^2}, \quad \theta_2 = \arctan2(z, r) - \arctan2(l_3\sin\theta_3, l_2 + l_3\cos\theta_3)
   \]  

3. **求解 \(\theta_3\)**（绕末端轴旋转）：  
   \[
   \theta_3 = \arccos\left( \frac{r^2 + z^2 - l_2^2 - l_3^2}{2l_2l_3} \right)
   \]

## 3. **4自由度SCARA机械臂**

### 3.1 解析法公式（R-R-P-R构型）

**末端位姿**：\( (x, y, z, \phi) \)（\(\phi\)为末端绕Z轴旋转角）  

**逆解步骤**：  

1. **平面位置求解**（前三关节）：  
   \[
   \theta_1 = \arctan2(y, x)  
   \]  
   \[
   \theta_2 = \arccos\left( \frac{x^2 + y^2 - l_1^2 - l_2^2}{2l_1l_2} \right)
   \]  

2. **垂直平移量 \(d_3\)**：  
   \[
   d_3 = z_{\text{目标}} - z_{\text{初始}}
   \]  

3. **末端旋转 \(\theta_4\)**：  
   \[
   \theta_4 = \phi - (\theta_1 + \theta_2)
   \]

## 4. **6自由度机械臂（满足Pieper条件）**

### 4.1 位置与姿态分离法（PUMA构型）

**Pieper条件**：前三关节轴相交于腕点 \( W \)，后三轴构成球形腕  

**逆解步骤**：  

1. **求解腕点位置 \( W \)**：  
   由末端位姿矩阵 \( T_6^0 \) 提取位置向量 \( o_6 \)，并利用腕部偏移量 \( d_6 \)：  
   \[
   W = o_6 - d_6 \cdot R_6^0 \cdot [0, 0, 1]^T
   \]  

2. **前三关节角求解**（几何法）：  
   - \(\theta_1 = \arctan2(W_y, W_x)\)  
   - \(\theta_2 = \arctan2\left( W_z - d_1, \sqrt{W_x^2 + W_y^2} \right) - \arcsin\left( \frac{d_3}{r} \right) \)  
   - \(\theta_3 = \arccos\left( \frac{r^2 + (W_z - d_1)^2 - a_2^2 - a_3^2}{2a_2a_3} \right) \)  

3. **后三关节角求解**（欧拉角分解）：  
   利用旋转矩阵 \( R_3^6 = (R_3^0)^T \cdot R_6^0 \)，分解为Z-Y-Z欧拉角：  
   \[
   \theta_4 = \arctan2(R_{23}, R_{13})  
   \]  
   \[
   \theta_5 = \arctan2\left( \sqrt{R_{13}^2 + R_{23}^2}, R_{33} \right)  
   \]  
   \[
   \theta_6 = \arctan2(R_{32}, -R_{31})  
   \]

## 5. **7自由度冗余机械臂**

### 5.1 伪逆雅可比法（SRS构型）

**解析-数值混合公式**：  

1. **前六轴解析解**（若满足Pieper条件）：  
   按6自由度方法求解 \( \theta_1 \sim \theta_6 \)，第七关节 \( \theta_7 \) 作为冗余自由度：  

2. **优化第七关节**：  
   通过最小化目标函数（如关节运动量）：  
   \[
   \min_{\theta_7} \| \theta_7 - \theta_{7,\text{初始}} \|^2  
   \]  
   约束条件：雅可比矩阵零空间投影：  
   \[
   \Delta \theta_7 = (I - J^\dagger J) \nabla H(\theta_7)
   \]  
   其中 \( J^\dagger \) 为雅可比矩阵的伪逆，\( H \) 为优化目标函数。

## 6. **Pieper条件的数学表述**

**适用情形**：六自由度机械臂需满足以下任一条件：  

1. **三轴相交**：存在三个连续旋转关节轴交于一点 \( W \)，即：  
   \[
   a_4 = a_5 = d_5 = 0 \quad (\text{简化DH参数})
   \]  

2. **三轴平行**：三个连续旋转关节轴平行，即：  
   \[
   \alpha_3 = \alpha_4 = \alpha_5 = 0 \ \text{或} \ \pi
   \]  

---

**注**：以上公式需结合具体DH参数调整，实际应用中需验证解的存在性及物理可行性（如关节限位）。
