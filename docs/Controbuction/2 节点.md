# 节点

节点是逻辑的单元。

## 节点特征

所有节点都是实现了 `Node` trait 的 Rust 结构体。具体的来说，需要实现如下的 trait：

```rust
#[async_trait::async_trait]
pub trait Node: Send + Sync {
    type Input;
    type Output;
    type Error: Debug + Send;

    async fn process(&mut self, input: Self::Input) -> Result<Self::Output, Self::Error>;
    async fn on_init(&mut self) -> Result<(), Self::Error>;
    async fn on_shutdown(&mut self) -> Result<(), Self::Error>;
}
```

当然在一般情况下 `on_init` 和 `on_shutdown` 这两个函数可以不实现，默认实现为返回 `Ok(())`。节点的输入输出类型可以是任意类型，但是需要满足 `Send` 和 `Sync` 的要求，以便于在多线程环境下使用。节点的 `process` 函数是节点的核心函数，负责处理输入并产生输出

节点本质上就是一个异步处理单元，接收输入数据，经过处理后输出结果。节点的初始化和关闭也被纳入生命周期管理。
